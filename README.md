# Icarus1216.github.io

这是参考 `https://chengyufang.tech/` 风格制作的静态学术个人主页。

## 查看

直接打开：

```text
personal-homepage/index.html
```

如果 IDE 支持预览，右键 `index.html` 选择 Preview / Open Preview。

## 修改

- `index.html`：替换姓名、学校、邮箱、经历、论文、获奖等内容。
- `assets/profile.svg`：替换成真实头像，文件名保持一致即可。
- `assets/pub-*.svg`：替换成论文图或项目图。

## 部署到 GitHub Pages

推荐仓库名：

```text
Icarus1216.github.io
```

推送到该仓库的 `main` 分支后，GitHub Pages 会发布到：

```text
https://Icarus1216.github.io/
```
